# webpack-learn

### 使用的插件包括：  
html-webpack-plugin  
clean-webpack-plugin  
copy-webpack-plugin  
splitChunks  
terser-webpack-plugin  
mini-css-extract-plugin  
optimize-css-assets-webpack-plugin  
webpack-merge  
webpack-bundle-analyzer  
### 使用的loader包括：  
babel-loader  
css-loader  
sass-loader  
url-loader  
eslint-loader
